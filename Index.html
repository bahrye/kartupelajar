Panduan Upload Foto

Agar foto otomatis masuk ke data siswa yang tepat, ikuti langkah berikut:

  1. Siapkan semua file foto siswa (JPG/PNG). Ukuran rekomendasi foto yaitu 3x4.
  2. Ubah Nama File (Rename) foto sesuai dengan NISN masing-masing siswa.
    Contoh: Jika NISN siswa adalah 00123456, nama file harus 00123456.jpg
  3. Pilih semua foto tersebut, lalu jadikan satu file ZIP (Compressed).
  4. Upload file ZIP tersebut melalui tombol di bawah ini.
Tips: Pastikan NISN di data Excel/Sistem sudah sesuai dengan nama file foto.
Panduan Import Excel

Ikuti langkah berikut untuk memasukkan data siswa secara massal:

  1. Download Template Excel
  2. Isi data siswa pada file Template tersebut.
    PENTING:
    • Format NISN harus Teks (gunakan tanda kutip satu ' di depan jika berubah jadi angka ilmiah).
    • Format Tanggal Lahir: Gunakan format Date Excel atau tulis Teks YYYY-MM-DD.
  3. Simpan file, lalu upload melalui tombol di bawah.
